
# Type of loop in javascript
- while loop
- do/while loop
- for loop
- for/in loop (object)
- forEach (array) 

While loop
---
```bash
var a = 1;
while (a<=10) {
  document.write("yes")
  a=a+1;
}
```

Do while loop
---
```bash
var a = 1;
do {
  document.write(a + "hello baba<br>");
  a=a+1
} while (a<=10>)

```
For loop
--
```bash
for (var a = 1;a <= 10; a++){
  document.write("hello")
}

```
